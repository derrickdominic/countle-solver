# countle_solver

Solver for puzzles on countle.org.

Usage:

    $ python3 countle_solver.py <target> <input1> <input2> ... <input6>

Example:

    $ python3 countle_solver.py 660 100 9 6 2 8 5
    229 solution(s) found.
    
    100 + 9 = 109
    109 + 8 = 117
    117 - 5 = 112
    112 - 2 = 110
    110 * 6 = 660
    
    ...
